Linux Bash Réseau Surveillance

Script de surveillance réseau

Script Bash réalisé sous Linux via WSL permettant de tester la disponibilité de plusieurs machines. Le script envoie un ping à chaque adresse et affiche le résultat avec un code couleur.

Pourquoi ce projet ?

Ce projet s’inscrit dans la continuité du script de diagnostic Linux. L’objectif est de surveiller rapidement plusieurs machines ou adresses réseau. Ce type de script peut aider un technicien à repérer les machines actives et celles qui ne répondent pas.

Ce que fait le script

📋 Liste de machines

Le script utilise un tableau Bash contenant plusieurs adresses IP et noms de domaine à tester.

🔁 Boucle for

La boucle for parcourt chaque machine de la liste une par une.

📡 Test avec ping

La commande ping -c 1 teste si chaque machine répond au réseau.

✅ Résultat du test

La variable $? permet de récupérer le résultat de la commande ping.

🟢 Affichage en vert

Si la machine répond, le script affiche qu’elle est active.

🔴 Affichage en rouge

Si la machine ne répond pas, le script affiche qu’elle est injoignable.
